How to use

This project is a Diabetes Prediction System built with FastAPI and a trained Machine Learning model.

  1. Enter patient details in the left form — e.g. Pregnancies, Glucose, Blood Pressure, BMI, Age.
  2. Click Predict to check if the patient is likely to have diabetes.
  3. The prediction result will appear below the form as Positive or Negative.

Note: This project is for educational purposes only and not a substitute for professional medical advice.
